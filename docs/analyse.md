# Objectif

CheckPoint est une application web permettant aux joueurs de gérer leur bibliothèque de jeux vidéo.

L'application permet de centraliser tous les jeux qui intéressent un utilisateur, qu'ils soient terminés, en cours, abandonnés ou prévus.

Pour chaque jeu, l'utilisateur peut consulter ou ajouter des informations personnelles (note, commentaire, statut...) afin de garder une vue d'ensemble de sa collection.
